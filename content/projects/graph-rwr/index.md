This project explores the use of random walk with restart (RWR) on tissue neighbourhood graphs to define cell–cell communication in the spatial transcriptomics context.

Key contributions:
- **Synthetic data generator**: Simulation of ligand-receptor signalling for benchmarking
- **Practical methods**: Scalable RWR-based inference of communication patterns in spatial transcriptomics data
